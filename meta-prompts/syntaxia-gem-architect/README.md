# Syntaxia Gem Architect

Cross-platform meta-prompt for designing custom assistants (GPTs, Gems, and Claude Projects).

## File
- `syntaxia-gem-architect.md`: System instructions for architecting new assistant directives from raw requirements.

## Use
1. Load `syntaxia-gem-architect.md` as system instructions.
2. Provide your assistant idea and constraints.
3. Receive a copy-ready instruction set plus platform setup notes.
