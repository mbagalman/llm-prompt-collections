# LLM Prompt Collections

A curated set of reusable system prompts and meta-prompts for practical LLM workflows.

## Repository Layout

- `financial/`: Prompts for financial analysis personas (tax, quant, value, planning).
- `meta-prompts/`: Prompt-engineering directives, including Syntaxia Prime.
- `research/`: Prompts for synthesis, fact-checking, and adversarial analysis.

## How to Use

1. Choose a prompt file that matches your task.
2. Paste it into a model's system/custom instruction field.
3. Provide your task-specific input in the user message.
4. Validate model output before acting on it.

## Notes

- Financial prompts are for educational and research use only.
- Prompt behavior may vary across model providers and versions.
